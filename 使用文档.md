# 🐋 Whale Tracker Bot · 使用文档

本文档分两部分：
- **第一部分 · 运营者（你）**：如何把 Bot 搭起来、配好收款、上线运行。
- **第二部分 · 终端用户**：你的客户在 Telegram 里怎么用这个 Bot。

如果你只想快速跑起来，照着第一部分从头做一遍即可。

---

## 第一部分 · 运营者部署指南

### 1. 这个 Bot 是做什么的

实时监控链上「鲸鱼钱包」的大额转账，并通过 Telegram 推送给订阅用户。支持 Ethereum / BSC / Tron 三条链，内置：

- 大额转账实时推送（金额阈值可配置）
- 聪明钱 Alpha 信号（代币热度榜 + 聪明钱排行）
- 每日定时 Alpha 日报
- USDT (TRC20) 链上自动验证收款，自动开通付费

商业模式：免费试用 3 天 → 引导付费（默认 5 USDT/月）。

### 2. 环境要求

- Python 3.10 及以上（推荐 3.11）
- 能联网的机器（本地或云服务器）
- 一个 Telegram 账号

### 3. 准备必填材料

| 材料 | 在哪拿 | 必填 |
|------|--------|------|
| Telegram Bot Token | 在 Telegram 找 [@BotFather](https://t.me/BotFather)，发 `/newbot` 按提示创建 | ✅ |
| USDT 收款地址 | 你自己的钱包地址，推荐 Tron TRC20（手续费低） | ✅ |
| TronGrid API Key | https://www.trongrid.io 免费注册 | 强烈建议 |
| Etherscan API Key | https://etherscan.io/apis 免费注册 | 强烈建议 |
| BSCscan API Key | https://bscscan.com/apis 免费注册 | 强烈建议 |
| 你的 Telegram 数字 ID | 找 [@userinfobot](https://t.me/userinfobot) 查询 | 建议（用于群发） |

> 不填链上 API Key 也能启动，但对应链的扫描会很容易被限流或失败，强烈建议都申请（全部免费）。

### 4. 安装与配置

```bash
# 进入项目目录
cd wallet-monitor-bot

# 1. 安装依赖
pip install -r requirements.txt

# 2. 复制环境变量模板
copy .env.example .env      # Windows
# cp .env.example .env      # Linux / Mac

# 3. 用编辑器打开 .env，至少填入 TG_BOT_TOKEN 和 PAYOUT_WALLET
```

`.env` 关键项最少填两个：

```ini
TG_BOT_TOKEN=从BotFather拿到的token
PAYOUT_WALLET=你的USDT收款地址
```

其余可调参数（定价、试用天数、推送阈值等）见下方「配置项速查表」。

### 5. 启动

```bash
python bot.py
```

看到下面这两行就说明启动成功：

```
✅ Background loops scheduled (Whale + Smart Money + Digest).
✅ Bot is running (Whale + Smart Money + Digest). Press Ctrl+C to stop.
```

然后去 Telegram 找到你的 Bot，发送 `/start` 测试。

### 7. 长期运行（7×24）

Bot 必须持续运行才能监控和推送。三种常见方式：

**方式 A · 云服务器 + systemd（推荐）**

创建 `/etc/systemd/system/whale-bot.service`：

```ini
[Unit]
Description=Whale Tracker TG Bot
After=network.target

[Service]
Type=simple
WorkingDirectory=/opt/wallet-monitor-bot
ExecStart=/usr/bin/python3 bot.py
Restart=always
RestartSec=10

[Install]
WantedBy=multi-user.target
```

```bash
sudo systemctl daemon-reload
sudo systemctl enable whale-bot
sudo systemctl start whale-bot
journalctl -u whale-bot -f      # 查看实时日志
```

**方式 B · 本地挂机**

```bash
# Windows: 保持终端开启
python bot.py

# Linux / Mac: 后台运行
nohup python3 bot.py > bot.log 2>&1 &
```

注意本地挂机时电脑不能休眠/关机。

**方式 C · Docker**

```bash
docker build -t whale-bot .
docker run -d --restart=always --env-file .env --name whale-bot whale-bot
```

### 8. 安全须知

- ⚠️ 切勿把 `.env`、`*.db` 提交到 GitHub（已在 `.gitignore` 排除）。
- ⚠️ Token / API Key 一旦泄露（贴聊天、截图、传仓库），立即去对应平台**吊销并重新生成**。
- Bot 通过 Telegram 官方 API 通信，本身不持有用户私钥，只读取公开链上数据。

## 第二部分 · 终端用户使用说明

把下面这段直接发给你的用户即可。

### 快速上手三步

1. 在 Telegram 打开 Bot，发送 `/start` 注册（自动获得 3 天免费试用）。
2. 用 `/add` 添加你想盯的钱包地址。
3. 之后只要该地址有大额转账，Bot 会自动推送给你。

### 命令清单

| 命令 | 作用 |
|------|------|
| `/start` | 注册 / 查看欢迎信息 |
| `/add <链> <地址> [标签]` | 添加追踪地址 |
| `/remove <地址>` | 移除追踪地址 |
| `/list` | 查看我追踪的地址 |
| `/status` | 查看账户状态（试用/付费、剩余天数） |
| `/pay` | 查看升级付费方式 |
| `/verify <付款地址>` | 付款后验证、自动开通付费 |
| `/alpha` | 查看聪明钱 Alpha 信号（热度榜 + 排行） |
| `/digest` | 查看今日 Alpha 日报 |
| `/smart` | 查看 Bot 正在追踪的聪明钱地址 |

### 添加地址示例

`<链>` 支持三个值：`eth`（以太坊）、`bsc`（币安链）、`trx`（波场）。

```
/add eth 0x28C6c06298d514Db089934071355E5743bf21d60 币安热钱包
/add bsc 0x... 某项目金库
/add trx TXFkJv3VRCg9LJhvyvLCfqxGVvq3vKTL5h 孙宇晨
```

标签是可选的，方便你自己辨认。地址格式：

- ETH / BSC：`0x` 开头，42 个字符
- Tron：`T` 开头，34 个字符

### 免费版 vs 付费版

| | 免费试用 | 付费 Pro |
|---|---|---|
| 追踪地址数 | 3 个 | 50 个 |
| 大额转账推送 | ✅ | ✅ |
| Alpha 信号 | 仅 Top3（脱敏） | 完整 Top10 + 聪明钱排行 |
| 有效期 | 3 天 | 30 天/次 |

### 如何升级付费

1. 发送 `/pay`，Bot 会显示收款地址（USDT TRC20）。
2. 用你自己的钱包给该地址转账对应金额的 USDT。
3. 转账成功后，发送 `/verify <你的付款钱包地址>`。
4. Bot 自动核对链上记录，验证通过即开通付费。

也可以直接把付款钱包地址发给 Bot，它会自动尝试验证。

### 常见问题

**Q：为什么没收到推送？**
A：① 该地址近期没有超过阈值的转账；② 试用/付费已过期（用 `/status` 查看）；③ 添加的地址链选错了。

**Q：`/verify` 提示未检测到支付？**
A：确认转账已上链确认、金额达标、用的是 TRC20 链，并且 `/verify` 后面跟的是你**付款的那个**钱包地址。

**Q：Alpha 信号看不全？**
A：完整 Top10 和聪明钱排行是 Pro 专享，免费版只显示 Top3，升级即可解锁。
