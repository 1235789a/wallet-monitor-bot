📖 1 分钟部署指南

① Fork 本仓库

克隆代码到你自己的 GitHub 账户

② 获取 Etherscan API Key

访问 etherscan.io/register 免费注册获取 API Key
（如需监控 BSC，同样去 bscscan.com/register）

③ 创建 Telegram Bot

在 Telegram 找 @BotFather，发送 /newbot 创建你自己的 Bot
拿到 Token 后，给自己发条消息，然后访问
https://api.telegram.org/bot<你的Token>/getUpdates 获取 Chat ID

④ 设置 GitHub Secrets

进入仓库 Settings → Secrets and variables → Actions
添加以下 3 个密钥：

TELEGRAM_BOT_TOKEN = 123456:ABC...
TELEGRAM_CHAT_ID = 987654321
ETHERSCAN_API_KEY = YourApiKey...
BSCSCAN_API_KEY = YourBscApiKey... # 可选

⑤ 配置监控地址

编辑 wallets.json，添加要监控的地址：

{
  "wallets": [
    {
      "address": "0xYourWalletAddress",
      "chains": ["ethereum", "bsc"]
    }
  ]
}

⑥ 提交代码

Push 到 GitHub 后，Actions 会自动开始运行。搞定！🎉
